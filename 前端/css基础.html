css- 层叠样式表


标签选择器

类选择器

ID选择器


我们一共三种css选择器



<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title>css的选择器</title>
		<style type="text/css">
			/* 这是标签选择器 */
			h5 {
				font-size: 20;
				color: coral;
			}
			
			/* 这是标签选择器 */
			.lei {
				font-size: 20;
				color: #7FFFFF;
			}
			
			/* 这是ID选择器 */
			#ID {
				font-size: 20;
				color: #7FFF00;
			}
			
		</style>
		
	</head>
	<body>
		<h5 class="h5"> 这是h5标签，我可以引用h5的标签选择器</h5>
		<p class="lei"> 这个是类选择器</p>
		<p id="ID"> 这是一个ID选择器 </p>
		
	</body>
</html>


如果想要在html中引入外部css，需要使用link标签



<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
  
  		<link rel="stylesheet" type="text/css" href="css/my.css"/>
  	</head>
  	<body>
  		<div class="blue"> 这是一个div </div>
  	</body>
  </html>
