安装pip

	curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
	python get-pip.py


安装pyenv

	参考：https://github.com/pyenv/pyenv/blob/master/README.md#installation
	安装一下依赖尤其是epel，紧接着安装python
	yum -y install wget gcc epel-release git 
	yum install bzip2-devel bzip2-libs -y
	yum install openssl openssl-devel openssl-static -y
	git clone https://github.com/pyenv/pyenv.git ~/.pyenv
	echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.bash_profile
	echo 'export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.bash_profile
	echo -e 'if command -v pyenv 1>/dev/null 2>&1; then\n  eval "$(pyenv init -)"\nfi' >> ~/.bash_profile
	重启shell
	pyenv install 3.6.1



安装virtualenv

pip install virtualenv



安装pyenv-virtualenv插件

参考：https://github.com/pyenv/pyenv-virtualenv

git clone https://github.com/pyenv/pyenv-virtualenv.git $(pyenv root)/plugins/pyenv-virtualenv

echo 'eval "$(pyenv virtualenv-init -)"' >> ~/.bash_profile


重启shell命令：exec "$SHELL"

创建一个项目目录/opt/simple，将此目录python环境设置成3.6.1


创建虚拟python环境

pyenv virtualenv 3.6.1 venv36

查看现在有的虚拟环境

[root@tingyun_server opt]# pyenv virtualenvs

3.6.1/envs/venv36 (created from /root/.pyenv/versions/3.6.1)

venv36 (created from /root/.pyenv/versions/3.6.1)

进入/opt 设置local python 环境

pyenv local venv36

之后进入目录就自动切换了
