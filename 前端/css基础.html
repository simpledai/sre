类选择器

ID选择器


我们一共三种css选择器



<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title>css的选择器</title>
		<style type="text/css">
			/* 这是标签选择器 */
			h5 {
				font-size: 20;
				color: coral;
			}
			
			/* 这是类class选择器 */
            /* 这是最常用的选择器 */

			.lei {
				font-size: 20;
				color: #7FFFFF;
			}
			
			/* 这是ID选择器 */
			#ID {
				font-size: 20;
				color: #7FFF00;
			}
			
		</style>
		
	</head>
	<body>
		<h5 class="h5"> 这是h5标签，我可以引用h5的标签选择器</h5>
		<p class="lei"> 这个是类选择器</p>
		<p id="ID"> 这是一个ID选择器 </p>
		
	</body>
</html>



如果想要在html中引入外部css，需要使用link标签

<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<!-- 引入css文件 -->
		<link rel="stylesheet" type="text/css" href="css/my.css"/>
	</head>
	<body>
		<!-- 引入css文件中的指定类 -->
		<div class="blue"> 这是一个div </div>
	</body>
</html>




层级选择器

这个其实不太常用，别人用，你知道是怎么回事就好



比如我有一段html，我只想让span的颜色发生一些改变

	<body>
		<div class="c1">
			<a href="">
				<div id="">
					这是一个验证
				</div>
				<span> 如果你想让这句话加粗 </span>
			</a>
		</div>
	</body>


我们可以定义一个ID选择器，然后span选用这个独立的ID

也可以这样定义一个选择器

中间用空格来区分

		<style type="text/css">
			.c1 a span {
				color: cadetblue;
			}
		</style>
